# Compatibilité Nextcloud

FileUni est conçu pour fonctionner avec les clients Nextcloud pour la gestion de fichiers, les favoris, le partage, la navigation média et l'accès via WebDAV.

## Portée actuelle

FileUni fournit les fonctionnalités de base pour la compatibilité avec les clients Nextcloud :

- Support du service WebDAV pour la synchronisation et l'accès distant
- Opérations de gestion de fichiers : parcourir, téléverser, télécharger, renommer, déplacer, copier, supprimer
- Favoris, fichiers récents et corbeille
- Prévisualisation des médias et des types de fichiers courants

Le comportement exact du client peut varier selon la configuration, les modules activés et la version du client Nextcloud.

## Statuts

- Disponible : la fonctionnalité est prête à l'emploi.
- En développement : la fonctionnalité est en cours de développement actif.
- Planifié : la fonctionnalité est prévue dans les prochaines versions.
