# Compatibilidad con Nextcloud

FileUni está diseñado para funcionar con clientes de Nextcloud para la gestión de archivos, favoritos, comparticiones, navegación de medios y acceso mediante WebDAV.

## Alcance actual

FileUni proporciona las funcionalidades básicas para la compatibilidad con clientes de Nextcloud:

- Soporte del servicio WebDAV para sincronización y acceso remoto
- Operaciones de gestión de archivos: explorar, subir, descargar, renombrar, mover, copiar, borrar
- Favoritos, archivos recientes y papelera
- Vista previa de medios y de tipos de archivo comunes

El comportamiento exacto del cliente puede variar según la configuración, los módulos habilitados y la versión del cliente Nextcloud.

## Lista

| Área | Objetivo | Estado |
| --- | --- | --- |
| Acceso WebDAV | Compatible con la conexión WebDAV del cliente Nextcloud | Disponible |
| Gestión de archivos | Explorar, subir, descargar, renombrar, mover, copiar, borrar | Disponible |
| Favoritos | Funcionalidades relacionadas con favoritos | Disponible |
| Comparticiones | Funcionalidades relacionadas con compartir | En desarrollo |
| Medios | Navegación y vista previa de medios | Disponible |
| Chat | Funcionalidad de chat | Planificado |
| Notes | Funcionalidad de notas | Planificado |

## Estados

- Disponible: la funcionalidad está lista para usar.
- En desarrollo: la funcionalidad está en desarrollo activo.
- Planificado: la funcionalidad está prevista para futuras versiones.
