# Возможности

На этой странице перечислены возможности, которые могут быть проверены из текущего репозитория и точек входа выполнения.

## Общее ядро для CLI и GUI

CLI-сервер и настольное приложение Tauri используют одну и ту же базовую библиотеку Rust:

- CLI — это основная точка входа сервера.
- GUI оборачивает тот же бэкенд с нативными для рабочего стола элементами управления сервисом, редактированием конфигурации и просмотром логов.
- Фронтенд для сервера статически встроен и обслуживается из `/`.

## Множественные пути доступа

Текущий проект предоставляет несколько способов доступа к одной и той же файловой платформе:

- Web UI по адресу `/`
- HTTP API по адресу `/api/v1/...`
- Документ OpenAPI по адресу `/api/v1/openapi.json`
- WebDAV, монтируемый по адресу `/@dav` по умолчанию (настраивается)
- Сервисы S3, FTP и SFTP при включении в конфигурации

## Возможности файловой платформы

Рабочая область содержит выделенный хаб хранилища VFS и фронтенд-модули для:

- Просмотра каталогов, загрузки, скачивания, перемещения, копирования, переименования и удаления файлов
- Поиска, истории, избранного, корзины и общих ресурсов
- Потоков предварительного просмотра файлов для распространённых типов файлов в Web UI
- Операций с архивами, таких как просмотр, сжатие и распаковка

Точная доступность некоторых функций зависит от вашей конфигурации и выбранного режима сборки.

## Модель развёртывания

Текущая модель выполнения намеренно явная:

- Конфигурация берётся из `config.toml`, а не из переменных окружения
- Каталоги выполнения разделены на пути конфигурации и данных приложения
- Отсутствующие обязательные поля конфигурации отклоняют запуск вместо молчаливого отката к значениям по умолчанию
- Установка сервиса сохраняет выбранные каталоги выполнения

## Операции и обслуживание

Текущие встроенные действия по обслуживанию включают:

- Режим настройки для конфигурации первого запуска
- Проверка конфигурации с помощью `--configtest`
- Установка, удаление, запуск, остановка, статус и перезагрузка системного сервиса
- Сброс пароля администратора из командной строки
- Экспорт и импорт резервных копий из командной строки

## Легковесное позиционирование

FileUni по-прежнему позиционируется как лёгкая файловая платформа. Публичный сайт в настоящее время описывает развёртывания с всего 32 МБ RAM, а сама рабочая область структурирована вокруг выборочного включения сервисов вместо принудительного включения каждой подсистемы в каждой установке.
