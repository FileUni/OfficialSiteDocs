# Совместимость с Nextcloud

FileUni поддерживает работу с клиентами Nextcloud для управления файлами, избранного, общего доступа, просмотра медиа и доступа через WebDAV.

## Текущий охват

FileUni предоставляет базовые функции для совместимости с клиентами Nextcloud:

- Поддержка сервиса WebDAV для синхронизации файлов и удалённого доступа
- Операции управления файлами: просмотр, загрузка, скачивание, переименование, перемещение, копирование, удаление
- Избранное, недавние файлы и корзина
- Предпросмотр медиа и распространённых типов файлов

Фактическое поведение клиента может отличаться в зависимости от конфигурации, включённых модулей и версии клиента Nextcloud.

## Список

| Область | Цель | Статус |
| --- | --- | --- |
| Доступ к WebDAV | Совместимость с WebDAV-подключением клиента Nextcloud | Доступно |
| Управление файлами | Просмотр, загрузка, скачивание, переименование, перемещение, копирование, удаление | Доступно |
| Избранное | Функции, связанные с избранным | Доступно |
| Общий доступ | Функции, связанные с шарингом | В разработке |
| Медиа | Просмотр и предпросмотр медиа | Доступно |
| Chat | Функция чата | Запланировано |
| Notes | Функция заметок | Доступно |

## Статусы

- Доступно: функция готова к использованию.
- В разработке: функция находится в активной разработке.
- Запланировано: функция включена в план будущих релизов.
