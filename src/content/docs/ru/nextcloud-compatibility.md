# Совместимость с Nextcloud

Эта страница описывает цель совместимости FileUni при использовании через клиенты Nextcloud.

Помимо работы на слабом железе и более высокой производительности на мощном оборудовании, совместимость с клиентами Nextcloud является еще одной важной особенностью, которую FileUni должен явно отражать во внешних материалах.

## Цель совместимости

FileUni стремится оставаться совместимым с основными сценариями клиентов Nextcloud, которых пользователи ожидают при миграции из среды Nextcloud.

Текущий фокус направлен на:

- Синхронизацию файлов и удаленный доступ через WebDAV
- Операции управления файлами в desktop- и mobile-клиентах
- Избранное, общий доступ, сценарии, связанные с медиа, и типичные взаимодействия с метаданными
- Сохранение привычных сценариев работы пользователей Nextcloud-клиентов

## Текущий охват

Текущий репозиторий уже содержит базовые элементы для этого направления:

- Поддержку сервиса WebDAV в storage-стеке
- Потоки Web UI и API для просмотра, загрузки, скачивания, переименования, перемещения, копирования, удаления, недавних файлов, избранного и корзины
- Предпросмотр медиа и распространенных типов файлов во frontend
- Модульные протокольные сервисы, которые можно включать в зависимости от масштаба развертывания

Эти части составляют основу совместимости с Nextcloud. Точное поведение клиента по-прежнему зависит от конфигурации, включенных модулей и конкретной версии клиента Nextcloud.

## Список

| Область | Цель | Статус |
| --- | --- | --- |
| Доступ к endpoint WebDAV | Совместимость с подключением Nextcloud-клиента по WebDAV | База доступна |
| Управление файлами | Просмотр, загрузка, скачивание, переименование, перемещение, копирование, удаление | Реализовано в текущей платформе |
| Избранное | Сохранять совместимость сценариев, связанных с избранным | Направление уже реализуется |
| Общий доступ | Сохранять совместимость сценариев, связанных с шарингом | Цель совместимости |
| Медиа | Сохранять работоспособность сценариев просмотра и предпросмотра медиа | Направление уже реализуется |
| Chat | Запланированная следующая возможность | Запланировано |
| Notes | Запланированная следующая возможность | Запланировано |

## Значение статусов

- База доступна: в репозитории уже есть необходимый протокольный или платформенный слой.
- Реализовано в текущей платформе: возможность уже существует в самом FileUni.
- Направление уже реализуется: связанная возможность уже есть, и работа по совместимости строится на ней.
- Цель совместимости: это часть заявленного направления продукта, но не обещание, что уже закрыты все крайние случаи всех клиентов.
- Запланировано: будущий объем, а не текущая функциональность.

## Рекомендация для внешних материалов

- FileUni — это легкая и масштабируемая файловая платформа.
- FileUni также проектируется так, чтобы оставаться совместимым с клиентами Nextcloud для управления файлами, избранного, общего доступа, медиа-сценариев и доступа на основе WebDAV.
- Chat и Notes относятся к следующему этапу roadmap и не должны описываться как уже выпущенные функции.
